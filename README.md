# 点到为止

一个基于 PySide6 和 pynput 的桌面连点器。程序支持普通连点模式，也支持矩形区域限制模式：开启矩形限制时，只有鼠标进入指定区域才会点击；关闭矩形限制时，就是普通的全局连点器。

## 功能

- 图形界面操作，不再依赖控制台输入。
- 全局快捷键切换开始和停止，默认首次运行是 `F8`。
- 快捷键可以在界面里重新设置。
- 点击频率使用“每秒点击次数”设置，范围是 `1-100 次/秒`。
- 支持全屏拖拽选择矩形区域。
- 矩形区域限制可以单独开启或关闭。
- 自动保存设置到 `settings.json`，下次启动会恢复上次的快捷键、点击频率、矩形开关和选区。

## 环境要求

- Python 3.9 或更高版本
- 依赖包：
  - `PySide6`
  - `pynput`

依赖已经写在 `requirements.txt` 中。

## 安装依赖

在项目根目录执行：

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
```

如果使用的是已有 Python 环境，也可以直接执行：

```powershell
python -m pip install -r requirements.txt
```

## 运行

在项目根目录执行：

```powershell
python main.py
```

## 使用方法

### 普通连点模式

1. 取消勾选 `启用矩形区域限制`。
2. 设置 `点击频率`，例如 `20 次/秒`。
3. 点击 `开始`，或按全局快捷键。
4. 鼠标当前位置会开始连续左键点击。
5. 再次点击 `停止`，或再次按全局快捷键，即可停止。

### 矩形区域限制模式

1. 勾选 `启用矩形区域限制`。
2. 点击 `选择区域`。
3. 在全屏半透明遮罩上按住鼠标左键拖拽，松开后保存矩形区域。
4. 点击 `开始`，或按全局快捷键。
5. 鼠标进入矩形区域后开始点击，移出区域后暂停点击。
6. 再次点击 `停止`，或再次按全局快捷键，即可完全停止。

选择区域时可以按 `Esc` 取消。

### 设置快捷键

1. 点击 `设置快捷键`。
2. 按下想要使用的按键或组合键，例如 `F6`、`F8`、`Ctrl + Alt + Space`。
3. 设置成功后，新快捷键会立即生效并保存。

录制快捷键时按 `Esc` 会取消本次设置。

## 配置文件

程序会在项目根目录生成或更新 `settings.json`。该文件保存本机运行偏好，不建议提交到仓库；公开仓库中提供了 `settings.example.json` 作为示例。

示例：

```json
{
  "clicks_per_second": 20,
  "toggle_hotkey": "<f8>",
  "region_enabled": false,
  "region": null
}
```

字段说明：

- `clicks_per_second`：每秒点击次数，程序会限制在 `1-100`。
- `toggle_hotkey`：全局开始/停止切换快捷键。
- `region_enabled`：是否启用矩形区域限制。
- `region`：矩形区域坐标；如果没有选区则为 `null`。

注意：如果 `settings.json` 已存在，程序会优先使用文件里的设置。因此界面显示的快捷键可能不是默认的 `F8`。

## 常见问题

### 启动时提示缺少 PySide6

执行：

```powershell
python -m pip install -r requirements.txt
```

### 快捷键没有反应

- 确认程序窗口仍在运行。
- 尝试换一个快捷键，例如 `F6`、`F8` 或 `Ctrl + Alt + Space`。
- 某些快捷键可能被系统、游戏或其他软件占用。

### 开启矩形限制后无法开始

开启 `启用矩形区域限制` 时，必须先点击 `选择区域` 并拖拽出一个有效矩形区域。

### 普通模式一开始就点到了按钮

从界面按钮启动普通模式时，程序会短暂延迟再开始点击。更推荐用全局快捷键启动和停止，这样鼠标可以提前放在目标位置。

## 注意事项

- 程序固定执行鼠标左键点击。
- 关闭窗口时会停止点击线程和全局快捷键监听。
- 连点器会对当前鼠标位置执行点击，使用前请确认鼠标所在位置安全，避免误操作。
